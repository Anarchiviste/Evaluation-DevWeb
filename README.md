# Évaluation Développement Web

## Principes : 

1) DRY (don't repeat yourself)
  - variables CSS

2) Responsivité et média queries :
  - tailles principales d'écran 
    - téléphone : 375px
    - Tablettes : 768px
    - PC : 1024px
    - Télévisions 1920px
  - Media queries 
    - règle qui permet d'appliquer des éléments seulement si certaines conditions sont remplies.

3) Bootstrap
  - breakpoints, modifier le comportement des éléments selon la taille de celui-ci.
